Download and try the fonts

To download and test the Topmarks fonts just use the links below. The fonts are supplied in both OpenType and TrueType formats. Please note that although the fonts do contain basic Latin character sets the current state of the fonts is still at a 'beta' stage. If you find any issues or wish to suggest changes, please email topmarks fonts
